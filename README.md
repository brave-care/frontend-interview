# Getting Started

Getting the frontend up and running:

```bash
cd frontend
yarn install
yarn dev
```

Getting the backend up and running:

```bash
cd backend
yarn install
yarn dev
```

## Tests

Each project has their own Jest unit tests.

```bash
yarn test --watch
```
